# References — {{项目名}}

> 本目录归档对项目有长期价值的**参考资料**：外部文档、设计/调研笔记、范例、链接清单。
> 目的：让 agent 不必每次重新上网检索，把"已验证有用"的知识沉淀在仓库内。

## 组织约定
- 一个主题一个文件，文件名用小写连字符，如 `auth-design.md`、`api-rate-limits.md`。
- 外链请附**抓取日期**与**一句话摘要**（外部内容会变，标注时效）。
- 大段外部原文不要整段粘贴，做**要点提炼 + 链接**即可（熵治理）。
